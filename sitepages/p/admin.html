How to add?

Step 1: Select a student

Step 2: Select task to be added for the selected student

Step 3: Click on ADD to add the selected task to the task list of selected student.

Step 4: Repeat steps 1, 2, and 3 and add all data to be submitted. You can see added data to be submitted in the table above which will appear as you add new data

Step 5: Click on submit and submit data( submit data only after adding all the data of all students you need to add currently )

Note: To view already completed tasks of a student, first select the name of the student from the input and click on view completed tasks button, which will appear then.
